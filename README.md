# MicroX App

## Requisitos

- python >= 3.12.0
- pip >= 23.2.1
- virtualenv >= 20.25.0


1. Crea un entorno virtual:

```
$ python -m virtualenv venv 
```

2. Entra al entorno virtual:

```
$ .\venv\Scripts\activate
```
3. Descargar paquetes:

```
$ pip install -r ".\requirements.txt"
```
